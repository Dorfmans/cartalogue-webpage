## Getting Started

Click [here](https://cartalogue-webpage.vercel.app/) to access the webpage

<br>

OR

<br>

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

<br>

## Technologies:

<br>

<ul>Next.js</ul>
<ul>ReactJs</ul>
<ul>Sass</ul>
<ul>Axios</ul>
